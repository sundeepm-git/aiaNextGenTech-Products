# Models

This directory contains GenAI model configurations, prompts, and related utilities for generative AI applications.

## What's Included

- **Prompt Templates**: Reusable prompt templates for various tasks
- **Model Configurations**: Settings and parameters for different LLMs
- **Fine-tuning Scripts**: Code for model fine-tuning and customization
- **Evaluation Tools**: Utilities for testing and evaluating model outputs

## Directory Structure

```
models/
├── prompts/        # Prompt templates and prompt engineering
├── configs/        # Model configuration files
├── evaluation/     # Model evaluation utilities
└── README.md       # This file
```

## Supported Models

This repository is designed to work with various LLM providers:
- OpenAI (GPT-4, GPT-3.5)
- Anthropic (Claude)
- Open-source models (Llama, Mistral, etc.)

## Usage

Check the `examples/` directory for sample code on how to use models with different providers.
